# سامانه تیکتینگ آفلاین (Backend API)

## پیش از اجرا

- در پوشه اصلی پروژه دستور `dotnet run` را در ترمینال اجرا کنید.
- دیتابیس `ticketing.db` به صورت خودکار ایجاد می‌شود.
- سامانه روی نشانی `https://localhost:7187` اجرا می‌شود.

## داده‌های اولیه (Seed data)

- **ادمین:** `admin@org.com` | رمز عبور: `Admin123`
- **کارمند:** `john@org.com` | رمز عبور: `User123`

## نکات مهم

- برای سادگی، ادمین می‌تواند تیکت را به هر کاربری که نقش Admin دارد اختصاص دهد.
- از کتابخانه `BCrypt.Net-Next` برای امنیت پسوردها استفاده شده است.
- تمام زمان‌ها به صورت UTC ذخیره می‌شوند.
